    public redirectLogin(redirectUrl: string, state: string): string source

    Redirects the user to player.me to authenticate your app. Once the user has accepted, they will then be redirected to the passed redirectUrl with whatever state you pass. The user will also be sent to the redirectUrl if they decline.

    Params:

    NameTypeAttributeDescription
    redirectUrl string

    The URL the user will be redirected to from player.me, once they've accepted/declined
    state string
    • optional

    A string that is passed back to the redirect URL

    Return:

    string

    The URL we're attempting to redirect to - in case automatic redirect fails

    Example:

    AuthService.redirectLogin(clientId, "https://example.com/foo", "bar");

    See:

    • http://docs.playermev2.apiary.io/#introduction/authentication/redirect-the-user-to-player.me
